Сделано Nenboard.
Эта программа запускает сервер для сайтов.

Как запустить:
1. Запишите ваш сайт в файл index.html в корневой папке программы.
2. Запустите программу через ярлык на рабочем столе, либо через консоль.
3. Если вы запустили программу через консоль, то укажите порт сервера в аргументах коммандной строки.
4. Все подключения пользователей будут выводиться в консоль, и в файл log.txt в корневой папке.
5. Все сообщения пользователей будут выводиться в консоль, и в файл messages.txt в корневой папке.
6. Вы также можете заблокировать пользователей. Для этого запишите IP адрес пользователя в файл ips.txt.

Как подключиться:
1. Запишите IP адрес в браузере и зайдите на него.
2. После этого сайт, загруженый на сервер будет виден на странице в браузере.
3. Если вы не можете зайти на сайт, то скорее всего у вас плохой интернет, либо вас заблокировали.

Для админов:
1. Чтобы отправить сообщение на сервер, используйте XMLHttpRequest.
3. Код запроса можно поменять, указав новый в аргументах коммандной строки.
4. Сообщения можно добавить в исключения, записав их в файл exceptions.txt в корневой папке.
